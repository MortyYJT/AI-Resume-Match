# AI Resume Match

AI Resume Match 是一个练手型 AI 简历岗位匹配项目。

第一阶段先做一个可演示 Demo：用户输入简历内容和岗位 JD，点击分析后，页面展示匹配分数、优势、短板和建议。

## 当前团队

| 成员 | 角色 | 当前负责 |
| --- | --- | --- |
| 后端 A | 负责人 / 后端 | 后端初始化、接口、联调推进 |
| 后端 B | 后端 | 跑通后端、熟悉结构、后续接 PDF |
| 前端 A | 前端 | 前端页面、接口调用、结果展示 |
| 产品顾问 | 产品建议 | 看方向、砍需求、给建议 |

## 当前技术方向

| 模块 | 选择 |
| --- | --- |
| 后端 | Java + Spring Boot |
| 前端 | 前端同学自选 |
| 数据库 | MySQL，第一期暂不接 |
| AI | 大模型 API，第一期暂不接 |
| 部署 | 先本地，后续再云端 |

## 第一期目标

```text
前端页面 -> 调后端 mock 接口 -> 后端返回固定报告 -> 前端展示报告
```

第一期不做：

- 登录注册
- MySQL
- PDF 解析
- 真实 AI 调用
- 历史记录
- Redis
- Docker
- 云端部署

## 第一期完成标准

本地打开前端页面，输入简历内容和岗位 JD，点击分析，页面展示一份匹配报告。

做到这个，第一期就算完成。
