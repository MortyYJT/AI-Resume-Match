# 需求说明

这份文档说明我们要做的产品是什么，以及第一期做到什么程度。

## 产品一句话

AI Resume Match 是一个 AI 简历岗位匹配工具。

用户输入简历内容和岗位 JD 后，系统给出匹配分数、优势、短板和优化建议。

## 目标用户

- 大学生
- 实习生
- 初级求职者
- 想根据岗位 JD 修改简历的人

## 最终完整流程

后续完整版本可以是：

```text
上传简历 -> 解析简历 -> 输入 JD -> AI 分析 -> 查看报告 -> 保存历史记录
```

但这不是第一期目标。

## 第一期流程

第一期只做：

```text
手动输入简历内容 -> 输入 JD -> 点击分析 -> 展示 mock 匹配报告
```

第一期先不上传 PDF，也不调用真实 AI。

## 第一期报告内容

页面需要展示：

- 匹配分数，例如 82
- 匹配等级，例如 高 / 中 / 低
- 优势
- 短板
- 简历优化建议
- 面试准备建议
- 投递建议

## 第一期不做

- 登录注册
- 用户权限
- PDF 上传和解析
- 真实大模型 API
- MySQL 保存
- 历史记录
- 管理后台
- 批量匹配

## 第一期验收标准

本地打开前端页面，输入简历内容和岗位 JD，点击分析后，页面展示一份结构完整的匹配报告。

只要这个流程跑通，第一期就完成。

## 后续再做

第一期完成后，再按顺序补：

1. PDF 上传
2. PDF 文本解析
3. 真实大模型 API
4. MySQL 保存报告
5. 历史记录
6. 登录注册
